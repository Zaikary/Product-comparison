# Matching

Решение задачи классификации товарных предложений (offers) и товаров из ассортимента маркетплейса (goods).
Задача — классифицировать пары offer-good и определить, является ли пара матчем (соответствием) или нет.

Этапы:
Анализ и предобработка данных
Feature engineering (работа с текстовыми и числовыми признаками)
Обучение классификационной модели
Валидация и оценка результатов
